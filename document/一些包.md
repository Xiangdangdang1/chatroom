### 用法

**写**

如果要生成

```json
{
  "name": "xdd",
  "age": 18,
  "skills": ["C++", "JS"]
}
```



```cpp
Json::Value root;

root["name"] = "xdd";
root["age"] = 18;

// 数组
root["skills"].append("C++");
root["skills"].append("JS");

//转为字符串(序列化)
Json::StreamWriterBuilder writer;
std::string jsonStr = Json::writeString(writer, root);

std::cout << jsonStr << std::endl;
```
